# [Sole by Sole](https://solebysole.com/)

## :sunny: 서비스 소개
나만의 지갑, 나만의 소품들이 필요하지 않으신가요?<br>
개성 있는 가죽 작품들을 이제 **솔바이솔레**에서 만나보세요.

<br>

## 🛠 핵심 기능
- 가죽 작품들을 구매할 수 있습니다.
- 장바구니에 상품을 담을 수 있습니다.
- 마음에 드는 작품이 없다면 작가에게 채팅을 통해 주문 제작을 요청할 수 있습니다.

<br>

## :computer: 기술 스택

### Back-end

- Spring Boot
